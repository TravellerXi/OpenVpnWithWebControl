欢迎使用畅游互联网OpenVPN系统

1. 注意：

本系统使用目的仅为学习（包括但不限于查找资料）。使用Openvpn首先需要有具有VPN配置文件。（本文创意模仿自中国科技大学:http://openvpn.ustc.edu.cn/）

2. OpenVPN是一个开放源码的基于SSL的VPN系统。OpenVPN开发站点在 www.openvpn.net。

它比PPTP VPN最大的优势在于仅仅使用UDP协议，支持从NAT设备后的连接。

3. 使用OpenVPN需要什么条件

3.1 使用VPN首先需要下载配置文件。

3.2 安装OpenVPN客户端

OpenVPN支持很多操作系统，请安装对应的客户端。

4. 安卓 使用

1. 安装openvpn安卓客户端

   下载安装文件(点此下载)。

   下载分发给你的.ovpn结尾的配置文件到手机并导入openvpn,（点击openvpn客户端右上角import from SD)  点击connect即可畅游互联网。

5. Windows 使用

1. 安装openvpn图形界面程序

   根据操作系统版本，下载其中之一的安装文件:




操作系统软件







Installer (32位操作系统), Windows XP

openvpn-install-2.3.10-I001-i686.exe









Installer (64位操作系统), Windows XP

openvpn-install-2.3.10-I001-x86_64.exe









Installer (32位操作系统), Windows Vista and later (Win7/Win8/Win10及以后)

openvpn-install-2.3.10-I601-i686.exe









Installer (64位操作系统), Windows Vista and later (Win7/Win8/Win10及以后)

openvpn-install-2.3.10-I601-x86_64.exe









   安装TAP驱动的时候有个没有经过数字签名的警告,选择允许安装。



2. 把分发给你的的以.ovpn结尾的配置文件下载到机器并拷贝到到openvpn安装目录config下面,一般是

   C:\program files\openvpn\config，也可能是C:\Program Files (x86)\OpenVPN\config
（如果安装时选择过路径，请去路径里找。或者右击桌面上的OpenVPN GUI图标，选择打开文件所在位置，向后再后退一位即可找到config目录）



3. 如果openvpn-gui程序已经启动，桌面右下角右击它选择退出



4. 到openvpn的安装目录bin下(默认是C:\program files\openvpn\bin)，或者在桌面上找到openvpn-gui.exe, 在这个openvpn-gui.exe文件上单击鼠标右键,选择"以管理员身份运行"(英语是Run as Administrator)



5. 此时右击桌面右下角openvpn，在右下角openvpn-gui上单击鼠标右键，单击"connect"可以连接vpn。

6. IOS苹果设备（包括MAC/IPAD/IPHONE）使用OpenVPN

IOS手机/Ipad设备使用OpenVPN，请注册美国区apple ID，然后登录该Apple ID，去apple store 搜索下载Openvpn客户端。然后下载系统分发的配置文件，用Openvpn打开即可完成导入。


MAC电脑请点此下载Openvpn客户端，安装完毕后，打开软件，下载系统分发的配置文件(在登陆系统里有下载配置文件)，直接将电脑里已下载好的以.ovpn结尾的文件拖动到该界面Configurations选项下的空白处。
具体做法请点此参考教程。备用教程链接。（以防被墙）
